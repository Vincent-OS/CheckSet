# Contribute to CheckSet
CheckSet is open to all contributions! You just need to respect the code of conduct.

## How to contribute?
You can contribute to CheckSet several ways:
- Software
- Documentation
- Translation

### Software
What you need to contribute to CheckSet:
1. Visual Studio 2022
2. Uno Platform
3. Windows Subsystem for Linux (WSL)

To contribute:
1. Fork this repository
2. On your forked repository, create a new branch for developping your work.
3. Do modification.
4. Always test your modification.
5. If all goods, send a Pull Request to your master branch on your forked repository.
6. Send a Pull Request to this repository and follow instructions.

### Translation
Help us to translate CheckSet in your language. You can follow the instructions on the Software section.
